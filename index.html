gotoma: console based pomodoros

gotoma is a command line utility for doing pomodoros. see the man page for more command line usage.

DEPENDENCIES

gotoma is written in Go, so you need to have a working Go installation. it also uses the plan9 mk(1) system.

TEXT-TO-SPEECH

the program looks for an executable file named "say" in the path, expecting that given the command line say some text, it will speak 'some text'. one is provided on os x; if you have festival (or don't mind installing festival), i've written a small [say program].

INSTALLING

  • to install to /usr/local: mk && sudo mk install

  • to remove:

    sudo mk uninstall

  • to install to ${HOME}:

    mk install PREFIX=${HOME}

  • to remove:

    mk uninstall PREFIX=${HOME}

BUGS

the installer won't install the man page properly in some cases (e.g. os x). this is ignored because os x ships with a say command.
